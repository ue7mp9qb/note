Fast and multi-purpose HTTP toolkit.

## 1. Install

安装

```
┌──(root㉿kali)-[~]
└─$ apt install -y httpx-toolkit
```

## 2. Usage

探测存活的站点

```
┌──(root㉿kali)-[~]
└─$ httpx-toolkit -l ~/urls.txt -sc -o ~/results.log
```
