Intelligent WHOIS client.

## 1. Usage

查询 Registrant Organization

```
┌──(root@kali)-[~]
└─$ whois example.com | grep "Registrant Organization"
```

通过域名查询注册信息

```
┌──(root@kali)-[~]
└─$ whois example.com
```

通过 IP 查询归属组织

```
┌──(root@kali)-[~]
└─$ whois 1.1.1.1
```
