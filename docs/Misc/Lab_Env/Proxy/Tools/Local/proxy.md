用于配置全局代理.

## 1. Install

安装

```
┌──(root㉿kali)-[~]
└─# curl -fsSL https://github.com/ue7mp9qb/script/raw/main/proxy_install.sh | bash
```

加载环境变量

```
┌──(root@kali)-[~]
└─$ source ~/.zshrc
```

## 2. Usage

配置代理

```
┌──(root㉿kali)-[~]
└─# proxy 10.0.2.2:10808
```

取消代理

```
┌──(root㉿kali)-[~]
└─# proxy
```
